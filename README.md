# Inside the Game Booking Portal, Full Working Prototype

This is a Vercel-ready Next.js prototype.

## What works
- Responsive web app for mobile, tablet and desktop
- Session booking flow
- Basket/cart
- Automatic 10% discount when booking 4 or more sessions
- Parent details form
- Player details form
- Booking confirmation screen
- Admin dashboard mock data
- PayPal payment instruction using marc.townsend123@btinternet.com

## Deploy to Vercel
1. Unzip this folder
2. Upload the folder to GitHub
3. Import the GitHub repo into Vercel
4. Deploy

## Local testing
npm install
npm run dev
